# dotfiles

I set these up by cloning this repo somewhere and then symlinking each file into my home directory:

```bash
cd ~

git clone git@github.com:mkcny/dotfiles.git

ln -s ~/dotfiles/bin
ln -s ~/dotfiles/.vim
ln -s ~/dotfiles/.vimrc
ln -s ~/dotfiles/.bash_profile
ln -s ~/dotfiles/.gemrc
ln -s ~/dotfiles/.gitconfig
```
